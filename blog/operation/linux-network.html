# Linux 网络配置

  • 在 /etc/sysconfig/network-scripts 下配置网络
  • /etc/sysconfig/network-scripts 下都是一些配置文件和脚本；ll 查看以 x 结尾的都是执行文件，用来启动和 停止网络的，但是不用这里面的命令启动停止网络。
  • ifcfg-lo：配置 127.0.0.1 的配置文件，不需要改动
  • 另外一个以 ifcfg 开头的文件是网络的配置文件
  • 重要配置项
    • BROADCAST：配置广播地址 192.168.0.255 标识全网广播，广播到该网段的所有计算机，255 标识该网段所有计算机。
    • NETWORK：配置网卡工作的网段地址 192.168.0.0，0 标识该网段。
    • DNS1：家用路由器都有 DNS 转发功能。配置成路由器地址，用来将 DSN 请求转发到相应的 DNS 服务器上。
    • IPADDR：Ip 地址，一般不需要改。
    • GATEWAY：默认网关，就是路由器地址。
    • ONBOOT：yes，标识电脑启动，该网卡被自动激活，才能上网。
    • BOOTPROTO：none / static：静态地址；dhcp：动态获取 ip 地址。电脑连上网，自动从路由器获取 Ip。
